# Sleep Script Patterns

Write for ears, not for eyes.

## Rules
- Keep sentences short.
- Use calm transitions.
- Repeat the core fact at least twice.
- Avoid dense lists.
- Prefer one knowledge unit per chunk.
- Keep the tone gentle and low-pressure.

## Vocabulary chunk template
1. Word
2. Meaning
3. Example sentence
4. Repeat the word + meaning
5. Pause cue

## Knowledge chunk template
1. Title
2. One-sentence takeaway
3. Brief explanation
4. Repeat the takeaway
5. Pause cue

## Good bedtime behaviors
- Prefer 5-15 minute lessons.
- Cap each script to a small number of chunks.
- End with a soft recap, not an energetic CTA.
