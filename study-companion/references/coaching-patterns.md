# Coaching Patterns

Use these patterns to keep the assistant useful during real study.

## Hint-first ladder

### Level 1 - orientation
- Restate the question briefly.
- Identify the tested concept.
- Give one starting hint.
- Ask the learner to try before revealing more.

### Level 2 - guided solving
- Split the problem into 2-4 steps.
- Explain why each step matters.
- Let the learner fill the final gap if possible.

### Level 3 - full solution
- Show the complete reasoning.
- Highlight the exact turning point or trick.
- Add one common mistake and one variation.

## OCR-specific behavior
- Quote the recognized question before deep analysis when confidence is mixed.
- Explicitly mark uncertain lines.
- Ask for a corrected photo or manual text supplement when the ambiguity changes the answer.

## End every session with one of these
- continue solving
- save to wrong-question log
- generate a mini quiz
- schedule review later
